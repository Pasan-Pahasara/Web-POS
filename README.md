# Web-POS
A JavaScript POS System made from SPA (Single-page application)

This is a small software creation made for uploading images using Spring MVC

###  `Languages`
* HTML
* Javascript
* CSS

###  `Frameworks`
* Boostrap Framework

### `Libraries`
* Sweet Alert
* jQuery
* Lombok
